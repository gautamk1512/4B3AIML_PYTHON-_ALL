# 4B3AIML_PYTHON-_ALL

This repository contains Python programming resources, solutions, and notes for input/output questions, syllabus topics, and practical exercises. 

## Contents
- `python_topics_notes.py`: Code notes and examples for all major Python topics (variables, control statements, functions, OOP, etc.)
- `python_io_questions_solutions.py`: Solutions for 30 classic Python input/output questions, organized by difficulty level.
- `extract_pdf_text.py`: Script to extract text from PDF files for study and review.

## How to Use
- Open `.py` files in your IDE to review code and run examples.
- Use `extract_pdf_text.py` to extract questions or notes from PDF study material.
